# 3-Axis Camera Gimbal

## Overview

Designed and developed a portable 3-axis brushless camera gimbal capable of stabilizing lightweight cameras using direct-drive BLDC motors and an embedded control system.

The project integrates mechanical design, embedded electronics, power management and hardware integration.
